# cuckoodo
Telegram task-manager/reminder for chats

# requirements
python 3.5+  
python-telegram-bot <https://github.com/python-telegram-bot/python-telegram-bot>  
mongoDB  

# Interface spec

`/add <issue> [@assignee]`

  Examples:

  `/add buy a bottle [@assignee]`

  `/add buy a bottle`

`/list [@assignee]`

  Examples:

  `/list`

  `/list @vasya`
